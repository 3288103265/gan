## StarGAN Estimator

Author: Wesley Qian

### How to run


1.  Run the setup instructions in `tensorflow_gan/examples/README.md`
1.  Run:

```
python stargan_estimator/train.py
```

### Description

An estimator implementation of StarGAN that abstracts away the training
procedure.
